Дано: имеется таблица Words в базе данных MySQL, содержащая сведения о всех существительных русского языка. Таблица имеет следующие атрибуты:
Word (varchar) – слово в ед. числе им. падеже.
Length (int) – количество букв в слове
AccentPosition (int) – позиция ударной гласной в слове

Задача: написать скрипт, получающий посредством GET-параметра слово в ед. числе и им. падеже, обращающийся к данной таблице и осуществляющий поиск хотя бы одной строгой рифмы к данному слову (т.е. окончания слов должны полностью совпадать, начиная с ударной гласной, если ударная гласная не последняя, и должны совпадать предыдущая буква и ударная, если ударная гласная последняя).
